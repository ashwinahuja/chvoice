### [L46 Project] Chrome Extension for Speech Enhancement

#### Install

```bash
pip install git+https://github.com/indrasweb/chvoice
```

```
.

│   ├── audio_processing.py   turn audio into spectrograms/images and visa-versa

    └── README.md
```
